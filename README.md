# Telegram AI Assistant (n8n)

An AI-powered Telegram bot built with n8n.

## Features
- Telegram Bot
- OpenAI GPT-5 mini
- Conversation Memory
- Tavily Web Search
- Google Sheets Logging

## Workflow
Telegram Trigger → Google Sheets → AI Agent → Telegram → Google Sheets

## Technologies
- n8n
- OpenAI
- Telegram Bot API
- Google Sheets
- Tavily
